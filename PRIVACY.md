# Rewind Privacy Policy

Rewind temporarily captures page HTML, visible links, visible media references, and styles so it can restore the page state after accidental navigation or reloads.

All captured data is stored locally in the user's browser using browser.storage.local.

Rewind does not collect, sell, transmit, upload, or share any page content, browsing history, URLs, screenshots, media, or personal data with any external server.

Snapshots are short-lived and automatically overwritten or pruned.
